My Notes

Main tag
Defines the primary content inside the document's body,
representing the page's central topic.
Img tag
Adds an image element to the page from a local folder
through the URL of the image.
Blockquote tag
Indicates that the text is a quoting on the page.
Cite tag
Adds a citation to a quote, when placed within the
blockquote tag.
P tag
Is a paragraph element on the page. Each tag should
represent only one paragraph.
